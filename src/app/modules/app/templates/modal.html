Parameters

type
Specify whether you want a confirm or prompt modal popup. (default: confirm)

model
Use this to hide the modal box again.

i18n (required)
Set the translations for this component

onConfirm
Specify the function which should be run when the user clicks Submit

onCancel
Specify the function which should be run when the user clicks Cancel

label
Use this to set what you want the confirmation button to say (default: OK)
