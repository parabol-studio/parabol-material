Parameters

placeholder
Used to specify the placeholder which will be used as the text input label (default: 'Select').

options
Import Collection with label and value to be used to construct the radio buttons. If no options are specified the default options will be a numeric range between 0 and 10.

keys
Specify the names of the keys, as a comma seperated list, which will be assigned to the label and value of the select options.
